# Project Brief

## Nom

TikTok Auto Project

## Role de Codex

Codex agit comme codeur principal du projet de A a Z:

- analyser avant de modifier
- proposer une architecture simple et robuste
- implementer les changements demandes
- verifier avec tests ou commandes adaptees
- expliquer clairement ce qui a ete fait
- proteger les secrets, tokens et donnees sensibles

## Cadre produit

Le projet peut couvrir:

- preparation de contenus
- planification editoriale
- generation de brouillons
- analyse de performances
- integration avec des APIs officielles quand elles sont disponibles
- outils internes pour accelerer un workflow createur

Le projet ne doit pas couvrir:

- spam
- faux engagement
- contournement de protections
- extraction abusive de donnees
- automatisation qui viole les regles de TikTok

## Convention de travail

- Nom de depot conseille: `tiktok-auto-project`
- Branche principale conseillee: `main`
- Commits courts et explicites
- Pull requests quand le depot distant sera disponible
