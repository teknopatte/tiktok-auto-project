Usage description

            

              TikTok Auto Project is a creator-owned web and desktop workflow used to prepare videos for the authorized TikTok account. The user signs in with TikTok Login Kit, selects or provides a video source, reviews the generated vertical short, chooses publishing options such as privacy and caption, and explicitly starts the export. The app then uses TikTok's official Content Posting API to publish the prepared video to the user's own TikTok profile.
            

            

              The app requests user.info.basic to identify the authorized creator account, video.publish to directly publish videos after user action, video.upload as a draft fallback when direct posting is not available, and video.list to reconcile published content and reporting inside the dashboard.
            

            

              No browser automation, scraping, credential sharing, or TikTok UI bypass is used. Tokens are stored locally on the user's machine, the creator can disconnect TikTok at any time, and the automatic runner is disabled by default until the user deliberately enables it.

Scope justification

            
    
              
  • user.info.basic: display the authorized creator account in the dashboard.
    • 
              
  • video.publish: publish user-approved videos through Direct Post API.
    • 
              
  • video.upload: upload as a draft fallback when direct publishing is unavailable.
    • 
              
  • video.list: verify published items and support reporting for the creator.
