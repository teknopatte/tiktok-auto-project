# Satisfying videos

Drop local MP4/MOV/MKV/WebM videos here.

The downloader picks one of these files at random when it renders TikTok-ready vertical shorts:

- top half: downloaded YouTube clip, kept horizontal with a blurred fill background
- bottom half: one satisfying video looped for the same duration

Examples: Trackmania gameplay, kinetic sand, soap cutting, slime, Minecraft parkour.
