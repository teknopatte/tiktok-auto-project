# PROJECT VISION

## Vision

Construire une plateforme locale d'automatisation TikTok capable de :

1. ingérer des vidéos ;
2. analyser leur contenu localement ;
3. détecter des passages candidats ;
4. mesurer objectivement chaque passage ;
5. sélectionner les meilleurs candidats ;
6. produire et optimiser les vidéos ;
7. publier via le pipeline existant ;
8. récupérer les performances accessibles ;
9. apprendre des résultats réels ;
10. améliorer les décisions futures.

## Philosophie

Le système ne doit pas essayer de deviner la viralité par intuition d'un LLM.

Il doit :

- mesurer ;
- stocker ;
- comparer ;
- tester ;
- apprendre.

## Contraintes

- local-first ;
- aucune API IA payante par défaut ;
- aucune note subjective de LLM ;
- architecture modulaire ;
- résultats explicables ;
- données brutes conservées ;
- tests obligatoires ;
- performances mesurées.

## Priorité stratégique actuelle

Construire le moteur objectif de détection et classement des meilleurs passages dans une vidéo longue.
